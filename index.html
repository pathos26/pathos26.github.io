About ColorTones

ColorTones는 음악과 조형예술의 경계를 허물고, 청각과 시각이라는 두 감각을 하나의 예술 경험으로 융합하는 창작 팀입니다.

우리는 소리가 만드는 형태와 색채, 그리고 조형이 담아내는 리듬과 하모니를 탐구하며, 서로 다른 전공과 감각 언어를 넘나드는 예술적 대화를 만들어갑니다.

Tones Meets Form. 이 슬로건은 우리가 추구하는 모든 것을 담고 있습니다. 음악의 톤(Tones)과 조형의 형태(Form)가 만나는 지점에서 새로운 예술적 가능성이 피어납니다.

Core Values

감각의 융합

청각과 시각이라는 다른 감각 언어를 하나의 경험으로 엮어내는 통합적 접근. 우리는 듣는 것과 보는 것 사이의 경계를 지우고, 두 세계가 하나로 만나는 공감각적 예술을 창조합니다.

상호 영감

음악이 형태를 만들고 형태가 소리를 낳는, 전공 간 경계를 넘는 창작적 대화. 서로 다른 분야의 예술가들이 영감을 주고받으며 새로운 창작의 지평을 열어갑니다.

공간의 재해석

소리가 공간을 채우고 조형이 시간을 담아내는, 예술 매체의 새로운 가능성 탐구. 우리는 공간과 시간, 물질과 비물질의 경계에서 예술의 새로운 차원을 발견합니다.
